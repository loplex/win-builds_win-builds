Win-builds - (Free) Software Packaging For Windows

Documentation/Downloads - Bug Tracker - Gitweb

Win-builds creates binary packages of libraries and tools for Windows. It is supported by the yypkg package manager. Both of them will become official tools and repositories for the mingw-w64 project.

  • targets windows 32bit, 64 bit soon
  • works from both windows and linux
  • GTK+, EFL, curl, libsoup, openssl, zlib, xz, ...
  • binary packages
  • 100% reproducible
  • build scripts available for everything, based on Slackware and SlackBuilds.org

Look at the Documentation for more information and downloads.
